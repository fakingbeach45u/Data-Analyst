üõí E-Commerce Sales Analysis

üìã Project Overview

Dalam project ini, saya menganalisis dataset e-commerce yang berisi 500,000+ transaksi dari tahun 2019-2023 untuk memahami pattern pembelian customer dan memberikan rekomendasi strategis untuk meningkatkan revenue.

Python Pandas NumPy Matplotlib Seaborn Jupyter Notebook

üéØ Objectives

  • Mengidentifikasi trend penjualan dan seasonal patterns
  • Analisis customer behavior dan purchasing patterns
  • Menentukan product categories dengan performa terbaik
  • Memberikan actionable insights untuk marketing team

üîç Methodology

  1. Data Collection & Cleaning
    • Import data dari multiple CSV files
    • Handle missing values (3.2% of total data)
    • Remove duplicates dan outliers
  2. Exploratory Data Analysis
    • Univariate & bivariate analysis
    • Time series decomposition
    • Customer cohort analysis
  3. Advanced Analytics
    • RFM (Recency, Frequency, Monetary) Analysis
    • Market basket analysis
    • Predictive modeling untuk future sales
