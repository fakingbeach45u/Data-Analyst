üìä Data Analyst Salary Analysis

üìã Project Overview

Proyek ini menganalisis data gaji Data Analyst dari berbagai sumber untuk memahami tren kompensasi, faktor-faktor yang mempengaruhi gaji, dan perbedaan regional dalam industri data analytics.

Python Pandas Matplotlib Seaborn NumPy

üéØ Key Analysis

  • Distribusi gaji berdasarkan lokasi geografis
  • Pengaruh pengalaman kerja terhadap kompensasi
  • Perbandingan gaji antar industri
  • Tren gaji dari waktu ke waktu

üíª Code Sample

import pandas as pd import matplotlib.pyplot as plt import seaborn as sns import numpy as np # Load dataset df = pd.read_csv('data_analyst_salaries.csv') # Basic info about dataset print("Dataset Info:") print(f"Shape: {df.shape}") print(f"Columns: {df.columns.tolist()}") # Data cleaning df = df.dropna() df['salary_in_usd'] = pd.to_numeric(df['salary_in_usd']) # Statistical summary print("\nSalary Statistics:") print(df['salary_in_usd'].describe()) # Visualization plt.figure(figsize=(12, 8)) sns.boxplot(data=df, x='experience_level', y='salary_in_usd') plt.title('Salary Distribution by Experience Level') plt.show() # Salary analysis by country country_salary = df.groupby('company_location')['salary_in_usd'].mean().sort_values(ascending=False) plt.figure(figsize=(15, 8)) country_salary.head(10).plot(kind='bar') plt.title('Average Salary by Country (Top 10)') plt.xticks(rotation=45) plt.show()

üîç Key Findings

  • Senior level analysts earn 40-60% more than entry level
  • US and European markets offer highest compensation
  • Remote work opportunities show competitive salaries
  • Tech industry leads in data analyst compensation

üõí E-Commerce Sales Analysis

üìã Project Overview

Analisis mendalam terhadap data penjualan e-commerce untuk mengidentifikasi pattern pembelian customer, seasonal trends, dan memberikan actionable insights untuk optimasi strategi bisnis.

Python Pandas Matplotlib Seaborn Plotly

üîç Analysis Focus

  1. Sales Performance Analysis - Revenue trends dan growth patterns
  2. Customer Behavior - Purchase patterns dan customer segmentation
  3. Product Analysis - Best performing categories dan products
  4. Geographic Analysis - Regional sales performance

üíª Code Sample

import pandas as pd import matplotlib.pyplot as plt import seaborn as sns from datetime import datetime # Load data df = pd.read_csv('ecommerce_sales.csv') # Data preprocessing df['order_date'] = pd.to_datetime(df['order_date']) df['month'] = df['order_date'].dt.month df['year'] = df['order_date'].dt.year # Monthly sales analysis monthly_sales = df.groupby(['year', 'month'])['total_sales'].sum().reset_index() # Top selling products top_products = df.groupby('product_name')['quantity'].sum().sort_values(ascending=False).head(10) # Customer analysis customer_stats = df.groupby('customer_id').agg({ 'total_sales': 'sum', 'order_id': 'count' }).rename(columns={'order_id': 'order_count'}) # Visualizations plt.figure(figsize=(15, 10)) # Monthly sales trend plt.subplot(2, 2, 1) monthly_sales.plot(x='month', y='total_sales', kind='line') plt.title('Monthly Sales Trend') # Top products plt.subplot(2, 2, 2) top_products.plot(kind='bar') plt.title('Top 10 Selling Products') plt.xticks(rotation=45) plt.tight_layout() plt.show()

üìà Key Insights

  • Q1 and Q4 show highest sales due to holiday seasons
  • Top 10 products contribute 40% of total sales
  • Customer segmentation reveals loyal repeat buyers
  • Region X outperforms others in sales growth
